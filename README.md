# AgentAppFlow

Open-source ecosystem to streamline iOS/macOS app creation with plugins, skills, automation, and tooling.

## Repositories
- [agentappflow-plugins](https://github.com/RINNECODER/agentappflow-plugins): plugin architecture and integrations.
- [agentappflow-skills](https://github.com/RINNECODER/agentappflow-skills): reusable developer skills and workflows.
- [agentappflow-macos-app](https://github.com/RINNECODER/agentappflow-macos-app): native macOS control center app.
- [agentappflow-cli](https://github.com/RINNECODER/agentappflow-cli): command-line automation and scaffolding tools.
- [agentappflow-docs](https://github.com/RINNECODER/agentappflow-docs): docs, architecture, and onboarding.

## Architecture Direction
- Plugin and skill runtime for composable development workflows.
- macOS app for orchestration and UX.
- CLI for scriptable automation and CI integration.
- Shared documentation and standards across repos.

## Branching Strategy
- `main`: stable baseline
- `dev`: active development
